# AlisaLang

### Структура
Программа обязательно должна содержать метод `main`. Он является входной точкой программы.
``` 
func main() {
  // logic
}
```

### Объявление переменной
Язык не содержит типы. Чтобы объявить переменную, вы должны написать ключевое слово `let`.
```
let foo = "bar";
foo = "foobar"
```
